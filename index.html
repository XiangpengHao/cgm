Export to Apple Health

HealthKit is iOS-only, but Apple's built-in Shortcuts app can log Blood Glucose — nothing to install. One-time setup:

valid readings, 1 per 5 min

The shortcut — experimental one-tap: download aidex-to-health.shortcut, enable iPhone Settings ▸ Shortcuts ▸ Advanced ▸ Allow Untrusted Shortcuts, then open the file to import. It's hand-built and untested, so if it won't import or logs nothing, build it manually instead:

  1. Get glucose-health.json (button above) onto your iPhone — AirDrop or Files / iCloud Drive.
  2. In the Shortcuts app, create a shortcut with these actions:
    • Get File → pick the JSON
    • Get Dictionary from Input
    • Repeat with Each → Get Dictionary Value glucose_mgdl; Get Dictionary Value time → Get Dates from Input; Log Health Sample (Blood Glucose · mg/dL · value · date)
  3. Run it and allow Health access. Re-run after each new export to add readings.

Uncalibrated CGM values — not for clinical use. Tip: after building it once, use the shortcut's share button to get an iCloud link you can reuse anywhere.
